## Installation

### Linux

* via ssh

[ ! -z "${BRANCH}" ] && [ ! -z "${PROJECT}" ] && git clone git@github.com:devmadinina/${PROJECT}.git -b ${BRANCH} ${PROJECT}/${PROJECT}.${BRANCH}

```sh
PROJECT="codingame"
BRANCH="master"
[ ! -z "${BRANCH}" ] && [ ! -z "${PROJECT}" ] && git clone https://github.com/devmadinina/${PROJECT} -b ${BRANCH} ${PROJECT}/${PROJECT}.${BRANCH}
```
